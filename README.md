Website Name: PlateShare

Live Site: https://shareurfoodtocommunity.netlify.app/

PlateShare is a platform that connects people who want to share or receive meals in their community. It aims to reduce food waste while helping those in need.

//Features

- User Authentication:** Secure login and registration system with avatar support.  
- Food Management:** Users can add, manage, and track the foods they want to share.  
- Food Requests:** Easily request available foods from other users.  
- Responsive Design:** Works seamlessly on desktop, tablet, and mobile devices.  
- Community‑Driven:** Connects donors and recipients locally, fostering a sharing culture.  

//Technologies Used

- React.js for the frontend  
- React Router for client‑side routing  
- Tailwind CSS for styling  
- Context API for state management  
- Firebase (or chosen backend) for authentication and data storage
